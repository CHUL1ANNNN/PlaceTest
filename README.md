# PlaceTest Monorepo

Базовая структура проекта (Task 0). Репозиторий содержит несколько доменов:

```
./
├─ docs/        # ТЗ, маппинг Авито, API сайта
├─ templates/   # темы/шаблоны
├─ app/         # оркестратор + UI
├─ automation/  # Dolphin + Avito
└─ data/        # локальные фото/логи (не коммитить)
```

## Общая схема потока

`сайт фото → app → AI → automation → Avito`

Где:
- **сайт фото**: источник исходных изображений.
- **app**: оркестратор и UI для управления процессом.
- **AI**: обработка/анализ контента.
- **automation**: автоматизация действий (Dolphin + Avito).
- **Avito**: публикация/обновление на площадке.
